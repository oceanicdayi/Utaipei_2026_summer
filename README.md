## 學員登記（最快路徑）

1. 開啟 https://script.google.com → 新增專案。
2. 貼上 `reference/gas/CourseRegistration.gs`。
3. 執行 `setupCourseRegistration` 並完成授權。
4. 從執行記錄取得表單填寫網址，分享給同學。

詳見 [學員資訊登記設定指南](reference/course_registration_form_guide.md)。
