# 北市大暑期地球科學課程計畫

**主題：地震學、地球物理與 AI／Agent 輔助教學實作**  
**時間：2026/7/17 起，每週五上午，共 6 週，每次 4 小時**  
**對象：中小學現職教師**  
**課程精神：教學相長、動手做、學習有用技能與知識，啟發興趣與創造力**

---

## 一、課程定位

本課程是為中小學現職教師設計的暑期地球科學課程，內容以**地震學與地球物理學**為主軸，同時導入 AI／Agent 工具與雲端平台，讓教師能把地震學知識轉化為可帶回學校使用的教材、互動網頁、資料分析活動與教學專題。

本課程也可作為開學後「地球物理學」課程的前導與素材庫。暑期課程偏重教師研習、動手做與教學轉化；開學後課程則可在相同素材基礎上加深理論、數學、資料分析與研究應用。

---

## 二、課程核心目標

1. 建立中小學教師對地震學與地球物理學的基本理解。
2. 學會使用真實地震資料進行簡單分析與教學設計。
3. 練習將地震學概念轉化成中小學可理解的活動。
4. 熟悉 opencode、Antigravity CLI、Hermes agent、GitHub、Hugging Face、Vercel、Google GAS、Google GWS 等工具。
5. 透過 Google GWS 與 GAS 建立教學自動化、資料收集與成果展示流程。
6. 每週完成一個可檢核的作業。
7. 安排一次野外實驗或簡易地震儀觀測活動。
8. 最後產出一份可帶回學校使用的地震教學作品。

# Week 1：課程介紹、帳號申請、軟體安裝與工作流程建立

**日期：2026/7/17**

## 本週定位

第一週不安排 Human Wave Demo。這週的重點是讓所有教師都能建立後續 5 週會使用的基本環境，包含帳號、軟體、雲端資料夾、GitHub repo、AI／Agent 工具與 Google GAS 測試。

## 本週內容

1. 課程總覽與學習目標說明。
2. 建立 Google Drive 課程資料夾。
3. 建立 GitHub 帳號與個人 repo。
4. 建立 Hugging Face 帳號。
5. 建立 Vercel 帳號。
6. 測試 Google Forms、Sheets、Apps Script。
7. 安裝或確認 Git、VS Code、Python、Node.js、Java。
8. 安裝或導覽 opencode、Antigravity CLI、Hermes agent。
9. 瀏覽 IRIS InClass 地震學教學資源。
10. 撰寫個人學習目標。

## 本週可檢核作業：作業 1

### 作業名稱

**完成個人數位學習工作區**

### 繳交內容

1. Google Drive 個人課程資料夾截圖。
2. GitHub 個人課程 repo 連結。
3. Hugging Face 個人帳號頁面截圖或連結。
4. Vercel 帳號建立完成截圖。
5. Google Form 測試表單連結。
6. Google Sheets 回應表截圖。
7. GAS 測試成功截圖。
8. 軟體安裝檢查表。

### 檢核標準

| 項目 | 通過標準 |
|---|---|
| 帳號 | Google、GitHub、Hugging Face、Vercel 至少完成 3 項 |
| GitHub | 有建立 repo，且包含 README.md |

# Week 2：地震儀、加速度、地震動與震度

**日期：2026/7/24**

## 本週主題

本週進入地震觀測的核心問題：**地震儀到底記錄了什麼？地面搖晃如何被量化？震度和加速度有什麼關係？**

## 教學活動

使用 IRIS 的 ground-shaking 類活動，讓學生透過三分量加速度計、手機、筆電或 USB 感測器理解地面搖晃與加速度測量。

## 本週可檢核作業：作業 2

### 作業名稱

**完成簡易震動觀測活動**

### 繳交內容

1. 一筆手機或感測器震動紀錄。
2. 一張 Google Sheets 圖表。
3. 一份學生版學習單。
4. 一份教師版教學說明。
5. 200 字說明：這個活動如何幫助學生理解震度？

### 檢核標準

| 項目 | 通過標準 |
|---|---|
| 資料 | 至少有一筆實測或模擬震動資料 |
| 圖表 | 能呈現震動強弱變化 |
| 學習單 | 至少有 3 題學生問題 |
| 教學說明 | 能清楚區分「感覺到的搖晃」與「儀器量測」 |

# Week 3：P-S 到時差、旅行時間曲線與震央定位

**日期：2026/7/31**

## 本週主題

本週回答一個地震學基本問題：**只靠幾個測站的地震波紀錄，如何知道地震發生在哪裡？**

## 教學活動

使用 IRIS 的震央定位活動。該活動讓學生從三分量地震紀錄中辨識 P 波與 S 波，量測 P-S 到時差，再用旅行時間曲線估算震央距，最後進行震央定位。

## 本週可檢核作業：作業 3

### 作業名稱

**完成震央定位教學範例**

### 繳交內容

1. 三個測站的 P 波與 S 波到時表。
2. P-S 到時差計算表。
3. 震央距估算結果。
4. 三測站定位圖。
5. GitHub 程式或 Google Sheets 連結。
6. 一份學生提問設計。

### 檢核標準

| 項目 | 通過標準 |
|---|---|
| P/S 到時 | 能正確標示至少三個測站資料 |
| 震央距 | 能由 P-S 到時差推估距離 |
| 定位圖 | 能畫出三個距離圓或等效定位圖 |
| 教學設計 | 能解釋為什麼至少需要三個測站 |

---

# Week 4：課本第 4 章精讀與真實地震波形資料

**日期：2026/8/7**

## 本週主題

本週將指定地震學課本 **Introduction to Earthquake Seismology 第 4 章** 納入課程。第 4 章作為理論深化週，連結前幾週的地震觀測、地震波與定位概念。

## 教學活動

使用 jAmaSeis 或其他即時地震波形工具，搭配真實地震事件資料，進行波形觀察、P/S 波辨識與教學轉化。

## 本週可檢核作業：作業 4

### 作業名稱

**完成第 4 章導讀與波形分析小報告**

### 繳交內容

1. 一頁第 4 章中文導讀。
2. 一張真實地震波形圖或截圖。
3. 標示 P 波、S 波或其他重要波相。
4. 三個可教給中小學生的概念。
5. 一段 200 字反思：哪些內容適合中小學？哪些適合開學後地球物理學？

### 檢核標準

| 項目 | 通過標準 |
|---|---|
| 導讀 | 能用教師可理解語言整理第 4 章重點 |
| 波形 | 至少有一張真實或教學用波形圖 |
| 標示 | 至少標出一個可討論的波形特徵 |
| 教學轉化 | 能區分「中小學版本」與「大學地球物理版本」 |

# Week 5：規模、震度、場址效應與建物反應

**日期：2026/8/14**

## 本週主題

本週處理地震教學中最容易混淆、也最貼近防災教育的問題：

- 地震規模和震度有什麼不同？
- 為什麼同一場地震，不同地方搖晃差很多？
- 為什麼建物與地盤會影響人的感受？

## 教學活動

使用 IRIS 的 Magnitude and Intensity 或 Earthquake Magnitude and Intensity 類資源，並搭配建物反應與場址效應案例。

## 本週可檢核作業：作業 5

### 作業名稱

**完成地震防災互動網頁初版**

### 繳交內容

1. GitHub Pages 或 Vercel 網頁連結。
2. 一張「規模 vs 震度」比較圖。
3. 一段場址效應說明。
4. 一個學生互動問題。
5. 一份教師教學提醒。
6. 一段 200 字防災教育應用說明。

### 檢核標準

| 項目 | 通過標準 |
|---|---|
| 網頁 | 可公開開啟或可由課程內部開啟 |
| 概念 | 能清楚區分規模與震度 |
| 場址效應 | 能說明不同地點搖晃差異 |
| 教學互動 | 至少有一題學生可回答的互動問題 |

## 七、課程一句話總結

這門課的目標不是只教地震學，而是讓中小學教師親手把地震學、地球物理、真實資料、AI agent、雲端平台與互動教材整合起來，做出能帶回學校、啟發學生興趣與創造力的地球科學教學作品。
